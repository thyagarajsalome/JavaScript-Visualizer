JavaScript Hoisting Visualization

Hoisting is JavaScript's default behavior of moving declarations to the top of their scope before execution. 
// This is like setting up the cups and ice beforehand (hoisted)
function makeLemonade() {
  console.log("Squeezing lemons and mixing with sugar!");
}

// This variable might have some leftover lemons from yesterday
var leftoverLemons = 2;

console.log("Let's make some lemonade! We have " + leftoverLemons + " lemons leftover.");
makeLemonade();  // You can call the function even before it's defined
